- Assembly
- BatchFile
- CSharp
- CSS
- HTML
- INIFile
- Java
- JScript
- [Lua](http://www.lua.org/about.html) (pronounced LOO-ah)
- MSIL
- Pascal
- Perl
- PHP
- PowerShell
- Python
- SQL
- VBDotNet
- VBScript
- XAML
- XML

Download it now from my [tools](http://tools.dunnhq.com/) page.
